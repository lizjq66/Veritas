This directory contains the artifact of Veritas's most recent simulated
trading session. These files are auto-generated by `pytest tests/test_loop.py`
and committed to the repository so anyone can see what Veritas actually
did without running anything.

- `journal.db` — the full SQLite journal (open with `sqlite3`)
- `events.jsonl` — step-by-step event stream
- `summary.md` — human-readable summary

Note: this is fake market data. The funding rates were synthesized by
the test fixtures in test_loop.py. What's real is the mechanism — the
Lean core, the assumption library updates, the exploration-to-exploitation
transition, and the deterministic reproducibility. Connecting Veritas
to a real market (Hyperliquid testnet) requires only replacing the
FakeObserver and FakeExecutor with their real counterparts in
python/observer.py and python/executor.py. All other code stays the same.
